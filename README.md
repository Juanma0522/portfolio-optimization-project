## Portfolio Performance

![Portfolio Performance](Figures/graph1)

This project applies Modern Portfolio Theory to construct and analyze an optimized portfolio composed of major technology companies.

## Methods used

- Covariance matrix analysis
- Efficient frontier
- Sharpe ratio optimization
- Monte Carlo simulation
- Backtesting against the S&P 500 (SPY)

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- yfinance

## Results

The optimized portfolio achieved strong long-term performance and significantly outperformed the SPY benchmark during the analyzed period (2010–2026).
